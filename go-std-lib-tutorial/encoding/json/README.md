# Modul: `encoding/json`

## Deskripsi Singkat
Package `encoding/json` mengimplementasikan encoding dan decoding JSON yang didefinisikan dalam RFC 7159.

## Cara Penggunaan
Silakan jalankan contoh kode di bawah ini dengan perintah:
```bash
go run main.go
```
