# Modul: `math`

## Ringkasan
Package `math` menyediakan konstanta dasar dan fungsi matematika.

## Penjelasan Lengkap (Fungsi & Tujuan)
Package `math` berisi banyak fungsi dan konstanta yang berhubungan dengan matematika dasar dan lanjutan.

**Tujuan dan Fungsi Utama:**
*   **Konstanta Penting:** Menyediakan nilai pasti untuk konstanta matematika seperti Pi (`math.Pi`) dan e (`math.E`).
*   **Fungsi Dasar:** Menghitung nilai absolut (`math.Abs`), akar kuadrat (`math.Sqrt`), atau pangkat (`math.Pow`).
*   **Pembulatan:** Membulatkan angka pecahan ke atas (`math.Ceil`), ke bawah (`math.Floor`), atau ke bilangan bulat terdekat (`math.Round`).
*   **Trigonometri:** Fungsi sinus, kosinus, tangen, dll (`math.Sin`, `math.Cos`, `math.Tan`).

**Mengapa menggunakan `math`?**
Diperlukan ketika Anda mengembangkan aplikasi yang membutuhkan kalkulasi presisi tinggi, geometri, statistik, algoritma fisika, atau bahkan sekadar membulatkan angka desimal dalam sebuah perhitungan harga pada aplikasi e-commerce.

## Cara Penggunaan
Silakan jalankan contoh kode di bawah ini dengan perintah:
```bash
go run main.go
```
*(Catatan: Anda harus berada di dalam direktori modul ini atau menyertakan path penuh file tersebut).*
