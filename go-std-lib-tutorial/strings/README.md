# Modul: `strings`

## Deskripsi Singkat
Package `strings` mengimplementasikan fungsi-fungsi sederhana untuk memanipulasi string UTF-8.

## Cara Penggunaan
Silakan jalankan contoh kode di bawah ini dengan perintah:
```bash
go run main.go
```
