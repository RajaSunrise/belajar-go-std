# Modul: `os/exec`

## Deskripsi Singkat
Package `exec` menjalankan perintah eksternal. Ini membungkus os.StartProcess untuk membuatnya lebih mudah di-remap.

## Cara Penggunaan
Silakan jalankan contoh kode di bawah ini dengan perintah:
```bash
go run main.go
```
