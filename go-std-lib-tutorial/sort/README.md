# Modul: `sort`

## Deskripsi Singkat
Package `sort` menyediakan primitif untuk mengurutkan slice dan koleksi yang didefinisikan pengguna.

## Cara Penggunaan
Silakan jalankan contoh kode di bawah ini dengan perintah:
```bash
go run main.go
```
