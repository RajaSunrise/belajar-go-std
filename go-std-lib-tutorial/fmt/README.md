# Modul: `fmt`

## Deskripsi Singkat
Package `fmt` mengimplementasikan I/O berformat dengan fungsi-fungsi yang mirip dengan `printf` dan `scanf` pada bahasa C.

## Cara Penggunaan
Silakan jalankan contoh kode di bawah ini dengan perintah:
```bash
go run main.go
```
